## 채식주의자를 위한 레시피 검색 서비스🥑   
Vegan recipe서비스는 다양한 재료로 만들 수 있느 요리레시피를 제공하기 위해 만든 서비스입니다.   
조금더 특별한 레시피를 제공하기 위해, 육류없이 만들 수 있는 레시피만 조회할 수 있는 서비스를 만들었습니다.   
Vegan recipe에서 서로 다양한 레시피를 등록, 공유하세요😀
